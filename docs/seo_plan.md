## 三、内链布局方案

### 首页内链策略
1. Hero 区 → `/divine`（主 CTA）
2. 场景卡片 × 5 → `/divine?scenario=career|love|wealth|study|health`
3. FAQ 区 → 各相关页面（卦典、占卜页）
4. 底部 SEO 段落 → `/hexagrams`（卦典）、`/about`（关于）
5. 信任模块 → `/hexagrams`（"基于《周易》经典"链接到卦典）

### 占卜页内链
1. 场景选择 → 各场景 deep link
2. 结果页 → 对应卦详情页 `/hexagrams/[id]`
3. 结果页 → "再占一卦" 回到 `/divine`
4. 结果页 → 相关卦推荐（互卦、错卦、综卦）

### 卦典页内链
1. 64卦列表 → 各单卦详情页
2. 单卦页 → 相关卦（上下卦、互卦、错综卦）
3. 单卦页 → "用此卦占卜" → `/divine`
4. 单卦页 → 回到卦典总览

### 全站内链规则
- 每个页面至少 3 个内链
- 首页获得最多反向内链（所有子页 footer 链回首页）
- 卦详情页之间形成网状结构（通过相关卦推荐）
- 面包屑导航：首页 > 卦典 > 乾卦

## 六、Google Search Console 优化建议

### 立即执行
1. **提交 sitemap.xml** — 确保包含所有语言版本的 URL（hreflang 标注）
2. **验证 hreflang 实现** — 当前 `getAlternateLanguages()` 已正确实现，确保每个页面都有
3. **提交 IndexNow** — 首页改版后立即通知 Google 重新抓取
4. **检查 Core Web Vitals** — LCP < 2.5s, FID < 100ms, CLS < 0.1

### 结构化数据
1. **FAQPage** — 首页 FAQ 区（上面的 JSON-LD）
2. **WebSite + SearchAction** — 站点级 schema（已有 HomeJsonLd）
3. **BreadcrumbList** — 卦典详情页面包屑
4. **HowTo** — "如何使用三币占卜法" 教程页

### 内容优化
1. **首页 H1** — 当前是"易经"两个字，太短。改为包含核心关键词的完整标题
2. **SEO 段落** — 当前两段 SEO 内容质量不错，建议增加内链锚文本
3. **图片 alt** — 所有装饰性图片加 alt 描述（太极图、云纹等）
4. **URL 结构** — 当前结构清晰（/divine, /hexagrams），保持不变

### 技术 SEO
1. **robots.txt** — 确保不阻止 Googlebot 抓取关键页面
2. **canonical** — 当前已实现，确保无重复页面
3. **页面加载速度** — 首屏 SSR，非关键 JS 延迟加载
4. **移动端适配** — Google Mobile-First Indexing，确保移动端体验优先
