## 每卦内容模板（目标2000-3000字/卦）

### 板块结构

```
1. 卦象概述（200-300字）
   - 卦名由来与字义
   - 上下卦构成（八卦组合）
   - 核心含义一句话总结

2. 卦辞详解（300-400字）
   - 原文引用
   - 白话翻译
   - 深度解读（结合《易传》《周易正义》）

3. 象辞详解（200-300字）
   - 原文引用
   - 白话翻译
   - 君子之道 / 处世智慧

4. 六爻逐爻详解（每爻150-200字，共900-1200字）
   - 爻辞原文
   - 白话翻译
   - 场景化解读（结合现代生活）
   - 吉凶判断

5. 现代应用指导（300-400字）
   - 事业：职场建议
   - 感情：关系指导
   - 财运：理财方向
   - 健康：养生提示

6. 互卦·综卦·错卦（100-150字）
   - 三种变换关系说明
   - 关联卦的核心提示

7. 历史典故（200-300字）
   - 与该卦相关的历史故事
   - 名人引用或典籍记载

8. 引用来源
   - 《周易》《易传·彖传》《易传·象传》
   - 《周易正义》（孔颖达）
   - 《周易本义》（朱熹）
```

### SEO 要求
- 每卦页面 H1："{卦名}卦 - 第{N}卦详解"
- 自然融入关键词：卦名+卦辞+爻辞+解读+含义
- 内链：爻辞中提到的相关卦加超链接
- 结构化数据：Article Schema + FAQ Schema（常见问题）
