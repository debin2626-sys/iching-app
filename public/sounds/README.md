# Sound Effects

This directory contains audio files for the divination experience.

## Required Files

| File | Description | Max Size |
|------|-------------|----------|
| `coin-throw.mp3` | Coin toss whoosh sound | <50KB |
| `coin-land.mp3` | Coin landing clink sound | <50KB |
| `complete-chime.mp3` | Completion chime sound | <80KB |
| `bg-guqin.mp3` | Background guqin (古琴) music loop | <500KB |
| `ambient-wind.mp3` | Ambient wind sound loop | <300KB |

## Notes

- All files should be MP3 format for maximum browser compatibility
- Keep file sizes small for fast loading
- Background sounds (`bg-guqin`, `ambient-wind`) should loop seamlessly
- One-shot sounds (`coin-throw`, `coin-land`, `complete-chime`) should be short and crisp
- Consider using 44.1kHz mono for smaller file sizes
